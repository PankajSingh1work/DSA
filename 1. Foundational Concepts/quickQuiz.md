# Quick Quiz

### 1. What’s the output of `print(groceries[2])` if `groceries = ["apples", "bananas", "milk"]`?

**Answer:** `milk`

---

### 2. Write a short program to print "Hello" 5 times using a loop.

**Answer:**

```python
for i in range(5):
    print("Hello")
```

---

### 3. If `age = 17`, what’s the output of:

```python
if age >= 18:
    print("Adult")
else:
    print("Minor")
```

**Answer:** `Minor`
